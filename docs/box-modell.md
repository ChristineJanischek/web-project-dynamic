# CSS Box-Modell

Jedes Element besteht aus:
```
+---------------------------+
|        margin             |
|  +---------------------+  |
|  |      border         |  |
|  |  +---------------+  |  |
|  |  |   padding     |  |  |
|  |  | +-----------+ |  |  |
|  |  | | content   | |  |  |
|  |  | +-----------+ |  |  |
|  |  +---------------+  |  |
|  +---------------------+  |
+---------------------------+
```
Eigenschaften:
- `margin`: Abstand zu anderen Boxen.
- `border`: Rahmen.
- `padding`: Innenabstand.
- `width/height`: Inhaltsgröße.

Praktisches Experiment: Nutze Browser DevTools, ändere Padding/Margin.
