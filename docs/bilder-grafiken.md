# Bilder & Grafiken

## Formate
- `.jpg` Fotos
- `.png` Transparenzen
- `.gif` einfache Animationen
- `.webp` modernes, oft kleiner
- `.svg` Vektor (skalierbar)

## Einbindung
```html
<img src="bilder/foto.jpg" alt="Beschreibung" width="300">
```
`alt` hilft Screenreadern & bei Ladefehlern.

## Responsiv
```css
img { max-width: 100%; height: auto; }
```
