# CSS Basis

CSS beschreibt wie Elemente aussehen.

## Einbindung
```html
<link rel="stylesheet" href="style.css">
```
oder Inline (nicht empfohlen):
```html
<p style="color:red;">Text</p>
```

## Selektoren
- `element` (z.B. `p`)
- `.klasse` (z.B. `.highlight`)
- `#id` (z.B. `#haupttitel`)

## Beispiel Datei `style.css`
```css
body { font-family: Arial, sans-serif; }
h1 { color: #0b5cad; }
.highlight { background: yellow; }
```
