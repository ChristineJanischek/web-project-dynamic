# Bilder-Ordner

Füge hier deine Bilder für die Galerie ein:
- 6-9 Bilder (z.B. `bild1.jpg`, `bild2.jpg`, etc.)
- Optional: Ein Hero-Bild `hero.jpg` (Landscape, min. 1920x1080px)

## Kostenlose Bildquellen

- [Unsplash](https://unsplash.com) - Hochwertige, lizenzfreie Fotos
- [Pexels](https://pexels.com) - Kostenlose Stock-Fotos
- [Pixabay](https://pixabay.com) - Freie Bilder & Videos

## Empfohlene Bildgrößen

- **Galerie-Bilder:** 800x600px (4:3) oder 800x800px (1:1)
- **Hero-Bild:** 1920x1080px (16:9)

## Platzhalter verwenden

Alternativ kannst du auch Platzhalter-Bilder verwenden:
- `https://picsum.photos/800/600` - Zufälliges Bild
- `https://via.placeholder.com/800x600` - Einfarbiger Platzhalter
