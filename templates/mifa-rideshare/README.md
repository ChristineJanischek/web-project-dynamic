# RideShare@School

Mitfahrgelegenheits‑App für Schulen: sicher, nachhaltig, effizient.

## Ziele
- Fahrten koordinieren, CO2 reduzieren
- Verifizierte Accounts, einfache Bedienung

## Tech‑Stack
- HTML/CSS/JS (Vanilla)

## Start
```bash
# lokal öffnen
$BROWSER index.html
```

## Aufgaben (MVP)
- Formular: Fahrt anbieten/suchen
- Liste: verfügbare Fahrten
- Validierung: einfache Checks
